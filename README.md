# VIP-Billionaires
React Native &amp; Firebase

## Getting Started

#### 1. Clone and Install

```bash
# Install dependencies
yarn install

# Install Pods
cd ios && pod install
```

#### 2. Open App in your simulator

Run this command to start the development server and to start your app on iOS simulator:
```
yarn run ios
```

Or, if you prefer Android:
```
yarn run android
```
